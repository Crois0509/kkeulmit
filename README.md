## 💡 주요 기능

### 1. push 알람
매일 사용자가 지정한 요일, 시간에 맞춰 날씨에 따라 AI가 옷차림을 추천해 줍니다.

### 2. 오늘의 옷, Color
Gemini AI가 날씨를 분석하여 옷과 색을 추천해줍니다.

### 3. 날씨 확인
오늘 날씨가 흐린지, 비가 오는지, 맑은 날인지 확인할 수 있습니다.
최저, 최고 및 평균 온도도 알려드려요
